Formulario de contacto multi-step con integración a Kommo CRM, almacenamiento en base de datos y exportación CSV.

== Description ==

FormFree es un plugin completo de formularios de contacto multi-step (hasta 5 pasos) con las siguientes características:

* **Multi-step forms**: Crea formularios de hasta 5 pasos con barra de progreso animada
* **Integración Kommo CRM**: Envía leads automáticamente a tu cuenta de Kommo
* **Base de datos**: Almacena todos los envíos en tu base de datos de WordPress
* **Exportación CSV**: Exporta todos los datos a CSV para análisis
* **reCAPTCHA v2**: Protección contra spam con Google reCAPTCHA
* **Validación en tiempo real**: Validación paso a paso para mejorar la experiencia del usuario
* **Notificaciones por email**: Email al admin y confirmación al usuario
* **Personalización de colores**: Colores personalizables para barra de progreso y botones
* **100% Responsive**: Optimizado para móviles y tablets
* **Compatible con Divi**: Funciona perfectamente con Divi y otros page builders
* **GDPR Compliant**: Retención configurable de datos

== Installation ==

1. Sube la carpeta `formfree` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a FormFree → Configuración para configurar el plugin
4. Crea tu primer formulario en FormFree → Crear Nuevo
5. Inserta el shortcode `[formfree id="1"]` en cualquier página

== Frequently Asked Questions ==

= ¿Cómo creo un formulario multi-step? =

1. Ve a FormFree → Crear Nuevo
2. Configura el nombre y el número de pasos (1-5)
3. Guarda el formulario
4. Usa el shortcode generado en cualquier página

= ¿Cómo integro con Kommo CRM? =

1. Ve a FormFree → Configuración
2. Activa la integración con Kommo
3. Ingresa tu subdominio, Client ID y Client Secret
4. Los leads se enviarán automáticamente

= ¿Cómo exporto los datos? =

Ve a FormFree → Envíos y haz clic en "Exportar a CSV"

= ¿Es compatible con Divi? =

Sí, 100% compatible. Inserta el shortcode con el módulo Code de Divi.
